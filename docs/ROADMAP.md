# NewFang Strategic Roadmap: From Nerve to Agency OS

## Vision
To build the definitive software execution system for agencies. A tool that doesn't just "clean docs," but drives project success through repeatable planning, deep repo-intelligence, and multi-agent orchestration.

---

## Phase 1: The Nerve (Current State)
*Core engine and local-first foundation.*
- [ ] **Scanner Engine**: Category-aware repo indexing.
- [ ] **Reconciliation Engine**: LLM-powered drift detection.
- [ ] **Agency Tool (Web)**: Local FastAPI dashboard with WebSocket chat.
- [ ] **Repo-Native Playbooks**: Skills versioned with the code.

## Phase 2: The Orchestrator (MCP & Deep Intelligence)
*Moving from "chatting" to "executing."*
- [ ] **MCP (Model Context Protocol) Integration**: Allow NewFang to act as a host or client for specialized tools (e.g., GitHub API, Linear, Slack).
- [ ] **Vector Memory Layer**: Persistent embeddings for documentation and code modules (ChromaDB/Qdrant).
- [ ] **Multi-Agent Orchestration**: Specialized workers for:
    - **The Auditor**: Continuous drift detection.
    - **The Architect**: Data model and integration mapping.
    - **The Spec-Writer**: Generating PRDs and User Stories from code evidence.
- [ ] **Outcome-Based Workflows**: Pre-built playbooks for "Onboarding," "Audit," and "Release Prep."

## Phase 3: The Desktop App (Tauri Transition)
*Native performance and professional aesthetics.*
- [ ] **Tauri-based Wrapper**: Convert the web dashboard into a lightweight, high-performance native desktop app.
- [ ] **System Tray Operation**: Background scanning and notifications for project drift or planning updates.
- [ ] **Local LLM Manager**: Built-in orchestration for Ollama/LM Studio with one-click setup.
- [ ] **Rich Aesthetic**: Professional dark/light modes, interactive graph views of project Spines.

## Phase 4: The Agency Standard (Repeatable Success)
*Features that drive project delivery excellence.*
- [ ] **Success Scorecards**: Benchmarking project health against industry/agency standards.
- [ ] **Snapshot & Diff**: Visualizing how a project's "Spine" has evolved over time.
- [ ] **Automated Documentation OS**: The "Cleanup" engine becomes proactive, suggesting re-organization as the repo grows.
- [ ] **Team Collaboration Mode**: Local-first syncing of .newfang/ playbooks across agency teams via Git.

## Phase 5: The Market Mover
*Commercializing the execution system.*
- [ ] **Portfolio View**: Single-pane-of-glass for managing 50+ projects simultaneously.
- [ ] **Risk Prediction**: AI-driven alerts for delivery risks based on code/doc drift patterns.
- [ ] **Agency "Nerve" Templates**: Sellable, repeatable planning frameworks for different project types (SaaS, Mobile, Web3).
